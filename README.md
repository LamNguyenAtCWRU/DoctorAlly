# How to run
## Step 1: clone or download the project
## Step 2: run in terminal: 
* npm i
* npm i react-router-dom
* npm install axios
* npm install mapbox-gl --save
* npm install @material-ui/core 
* npm install @material-ui/icons
* npm install @material-ui/lab
* npm i react-player
